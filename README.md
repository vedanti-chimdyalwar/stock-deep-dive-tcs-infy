# INFOSYS-TCS-FINANCIAL-ANALYSIS

## Overview

This repository contains a comparative financial analysis of **Infosys**
and **TCS (Tata Consultancy Services)** using annual reports and
financial statements. The project evaluates company performance through
balance sheets, profit & loss statements, cash flow analysis, and
financial ratios.

## Objectives

-   Compare the financial performance of Infosys and TCS
-   Analyze profitability, liquidity, efficiency, and solvency
-   Study trends using historical annual reports
-   Understand overall business growth and financial health

## Dataset / Sources

Financial data is collected from: - Annual reports (2022--2025) -
Balance sheets - Profit & Loss (P&L) statements - Cash flow statements -
Financial ratio reports
